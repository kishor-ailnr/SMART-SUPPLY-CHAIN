# 🌐 Nexus Logistics Intelligence Hub 2.0 (SMART-SUPPLY-CHAIN)

Nexus is a state-of-the-art, multi-modal logistics command center designed for real-time visibility, predictive analytics, and automated fleet management across all transport sectors.

## 🚀 The Ecosystem
The platform consists of five interconnected intelligence portals:

*   **🛰️ Nexus Hub**: The central neural node for cross-platform navigation and global overview.
*   **🚢 Seaways (Port 6004)**: Maritime vessel tracking with high-fidelity AIS streams and voyage prediction.
*   **✈️ Airways (Port 6002)**: Aviation intelligence featuring real-time flight radar and cargo manifest management.
*   **🚚 Roadways (Port 6001)**: Surface transport monitoring with vehicle telemetry and NHINET integration.
*   **🚆 Railways (Port 6003)**: National rail synchronization featuring track-snapped pathing and Digital Twin simulations.

## ✨ Key Features
*   **Digital Twin Synchronization**: High-fidelity 3D/2D simulations of fleet movements and operational environments.
*   **AI-Driven Predictive Analytics**: Real-time event prediction for weather disruptions, traffic congestion, and maintenance.
*   **Unified Command Center**: Seamless cross-portal navigation with a standardized "Back to Hub" architecture.
*   **Glassmorphism UI/UX**: Premium, modern interface designed for high-stakes logistics operations.
*   **Real-time WebSockets**: Low-latency data synchronization across all backends (Ports 6001–6004).

## 🛠️ Tech Stack
*   **Frontend**: React.js, Vite, Tailwind CSS, Framer Motion (for cinematic animations).
*   **Backend**: Node.js, Express, Socket.io (WebSockets).
*   **Intelligence**: Lucide Icons, Leaflet/MapLibre (Geospatial visualization).
*   **Infrastructure**: Environment-variable driven for seamless cloud deployment (Railway, Vercel, Render).

## 📥 Installation & Setup
1. **Clone the repository**:
   ```bash
   git clone https://github.com/kishor-ailnr/SMART-SUPPLY-CHAIN.git
   ```
2. **Install Dependencies** (for each folder):
   ```bash
   # Run in Hub, Roadways/frontend, Airways, Railways, Seaways/seaways/client
   npm install
   ```
3. **Launch the Hub**:
   ```bash
   cd Hub
   npm run dev
   ```
